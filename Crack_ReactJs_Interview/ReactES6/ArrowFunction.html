React ES6 Arrow Functions: Arrow Functions: Arrow functions allow us to write shorter function syntax: Example: Before ES6: hello = function() { return "Hello World!"; } hello(); //Hello World! ------------------------------------------------------------------------------------------------ With Arrow Function: Example: hello = () => { return "Hello World!"; } hello(); //Hello World! ------------------------------------------------------------------------------------------------ Arrow Functions Return Value by Default: Example: hello = () => "Hello World!"; hello(); //Hello World! Note: This works only if the function has only one statement. ------------------------------------------------------------------------------------------------ If you have parameters, you pass them inside the parentheses: Example: Arrow Function With Parameters: hello = (val) => "Hello " + val; hello("Golu"); //Hello Golu
